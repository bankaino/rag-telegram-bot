# Безопасность данных в Flowly

Мы понимаем, что задачи и проекты вашей команды — это конфиденциальная информация. Ниже описано, как Flowly защищает ваши данные.

---

## Шифрование

**В транзите:** все данные между браузером / мобильным приложением и серверами Flowly передаются по протоколу TLS 1.2 / 1.3. Незашифрованное соединение HTTP автоматически перенаправляется на HTTPS.

**В хранилище:** данные базы данных и файлы вложений хранятся в зашифрованном виде (AES-256). Ключи шифрования управляются через AWS KMS с ротацией раз в год.

---

## Инфраструктура

Flowly работает на облачной инфраструктуре AWS (регион eu-central-1, Франкфурт). Мы не используем серверы в юрисдикциях с обязательным раскрытием данных по запросу иностранных властей.

**Резервное копирование:** полный бекап базы данных выполняется ежедневно, инкрементный — каждый час. Резервные копии хранятся в зашифрованном виде 30 дней.

**Uptime:** для тарифа Business предусмотрен SLA 99,9% (не более 8,7 часов простоя в год). Статус и история инцидентов публично доступны на status.flowly.io.

---

## Управление доступом

**Разграничение ролей:** каждый участник рабочего пространства получает ровно те права, которые соответствуют его роли (Владелец, Администратор, Участник, Наблюдатель). Доступ к проектам настраивается отдельно: можно создать закрытый проект, видимый только определённым участникам.

**Двухфакторная аутентификация (2FA):** доступна для всех тарифов. На тарифе Business администратор может сделать 2FA обязательной для всей команды.

**SSO (Single Sign-On):** тариф Business поддерживает вход через Google Workspace и Microsoft Entra (Azure AD) по протоколу SAML 2.0 / OIDC. При увольнении сотрудника деактивация корпоративного аккаунта автоматически отзывает доступ к Flowly.

**Сессии:** сессии автоматически истекают через 30 дней неактивности. Принудительно завершить все активные сессии можно в «Профиль» → «Безопасность» → «Завершить все сессии».

---

## Приватность и GDPR

Flowly соответствует требованиям GDPR (Общий регламент защиты данных ЕС).

- Мы не продаём пользовательские данные третьим сторонам.
- Данные аналитики (Mixpanel) анонимизированы.
- Вы можете запросить экспорт всех своих данных или их удаление — напишите на privacy@flowly.io.
- Политика конфиденциальности: flowly.io/privacy

**Субпроцессоры:** Flowly использует ограниченный список субпроцессоров (AWS, Mixpanel, Intercom для поддержки). Актуальный список: flowly.io/subprocessors.

---

## Реагирование на инциденты

При обнаружении потенциального инцидента безопасности:
1. Уведомления о подтверждённых инцидентах направляются затронутым клиентам в течение 72 часов (как предписывает GDPR).
2. Мы публикуем post-mortem на status.flowly.io после устранения критических инцидентов.

**Ответственное раскрытие уязвимостей (Bug Bounty):** если вы обнаружили уязвимость, напишите на security@flowly.io. Мы ценим ответственное раскрытие и признаём исследователей на нашей странице благодарностей.

По вопросам безопасности, DPA (соглашение об обработке данных) и аудитам — пишите на security@flowly.io.
