# Интеграции Flowly

Flowly подключается к популярным инструментам, чтобы вы работали в привычной среде. Все интеграции настраиваются в разделе «Настройки» → «Интеграции».

---

## Telegram

**Тарифы:** Pro, Business

Подключите Telegram, чтобы получать уведомления о задачах прямо в мессенджере.

**Что приходит в Telegram:**
- Вам назначена задача
- Вас упомянули в комментарии (@имя)
- Приближается дедлайн (настраивается: за 1 день, за 1 час, в момент)
- Задача, в которой вы участник, изменила статус

**Как подключить:**
1. Перейдите в «Настройки» → «Интеграции» → «Telegram».
2. Нажмите «Подключить» — откроется ссылка на бот @FlowlyNotifyBot.
3. В боте нажмите /start — уведомления начнут поступать немедленно.
4. Чтобы настроить типы уведомлений, вернитесь в «Настройки» → «Уведомления».

## Slack

**Тарифы:** Pro, Business

Отправляйте уведомления о задачах и проектах в Slack-каналы.

**Возможности:**
- Личные уведомления (те же, что и в Telegram, но в личные сообщения Slack)
- Уведомления в канал: когда задача в проекте создана, завершена или просрочена
- Команды `/flowly list` и `/flowly task #ID` прямо из Slack

**Как подключить:**
1. «Настройки» → «Интеграции» → «Slack» → «Add to Slack».
2. Авторизуйте приложение Flowly в нужном Slack workspace.
3. Выберите канал для уведомлений по умолчанию.

## Google Calendar

**Тарифы:** Pro, Business

Синхронизируйте дедлайны задач Flowly с Google Calendar.

**Как работает:**
- Задачи с дедлайном автоматически появляются в Google Calendar как события (только чтение — редактировать их нужно в Flowly).
- Изменение дедлайна в Flowly обновляет событие в Calendar в течение 5 минут.
- Поддерживается подписка на ICS-ссылку (для интеграции с Apple Calendar, Outlook и другими).

**Как подключить:**
«Настройки» → «Интеграции» → «Google Calendar» → «Подключить аккаунт Google».

## GitHub

**Тарифы:** Business

Свяжите задачи Flowly с pull request'ами и issue на GitHub.

**Возможности:**
- Упоминание задачи Flowly в PR или commit (`flowly:#ID`) — статус задачи обновляется автоматически при мерже
- Автоматическое создание задачи в Flowly при открытии GitHub Issue (настраивается)
- Двусторонняя синхронизация статусов: закрытый PR → задача «Завершена»

**Как подключить:**
«Настройки» → «Интеграции» → «GitHub» → «Установить GitHub App» на нужный репозиторий или организацию.

## Jira

**Тарифы:** Business

Импортируйте задачи и проекты из Jira Cloud в Flowly, или используйте обе системы параллельно.

**Возможности:**
- Однократный импорт проектов и задач из Jira
- Двусторонняя синхронизация статусов задач (настраивается маппинг статусов)

**Как подключить:**
«Настройки» → «Интеграции» → «Jira» — потребуется API-токен Jira Cloud.

---

## REST API

**Тарифы:** Business

Полный REST API для интеграции Flowly с собственными системами.

**Возможности:**
- Создание, чтение, обновление и удаление задач, проектов, участников
- Webhooks: отправка событий на ваш URL при изменении задач
- Аутентификация через API-ключи (генерируются в «Настройки» → «API»)
- Документация: docs.flowly.io/api

## Импорт данных из CRM и других систем

**Тарифы:** Pro, Business

Если вы ведёте клиентов, сделки или задачи в другой системе — CRM, таблицах или таск-трекере — данные можно перенести в Flowly.

**Импорт из CSV**
Любые задачи и проекты можно загрузить через CSV-файл. Формат: каждая строка — одна задача с полями название, описание, исполнитель, дедлайн, приоритет, статус. Шаблон CSV доступен в «Настройки» → «Импорт» → «Скачать шаблон».

**Что обычно импортируют из CRM:**
- Список клиентов → проекты или задачи в Flowly (например, «Онбординг клиента X»)
- Сделки и этапы воронки → задачи с метками и статусами
- Контакты и ответственных → участники и исполнители задач

**Как импортировать:**
1. Экспортируйте данные из вашей CRM в CSV (эта функция есть в большинстве систем: amoCRM, Bitrix24, HubSpot, Salesforce).
2. Приведите файл к формату Flowly (используйте шаблон).
3. Перейдите в «Настройки» → «Импорт» → «Загрузить CSV».
4. Сопоставьте колонки вашего файла с полями Flowly.
5. Нажмите «Импортировать» — задачи появятся в выбранном проекте.

**Импорт через API**
На тарифе Business доступен полный REST API — можно автоматизировать перенос данных из любой системы, которая умеет делать HTTP-запросы. Документация: docs.flowly.io/api

**Нужна помощь с импортом?**
Если структура ваших данных нестандартная или объём большой — напишите на support@flowly.io, мы поможем настроить импорт индивидуально.
